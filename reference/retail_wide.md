# Sales data from an Australian Retailer in time series format

A dataset containing 42 products' sales

## Usage

``` r
retail_wide
```

## Format

An object of class mts (inherits from ts, matrix) with 333 rows and 43
columns.

This data set is the wide format of
[retail](https://akai01.github.io/caretForecast/reference/retail.md)
data.

## Source

<https://robjhyndman.com/data/ausretail.csv>
